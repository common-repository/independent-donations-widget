== Description ==

With the use of the [Sunlight API](http://sunlightfoundation.com/api/accounts/register/), this widget will show the most recently updated information regarding spending by Political Action Committees (PACs).

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `Independent Donations Widget` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the 'Independent Donations Widget' option in the Appearance->Widgets area and install in a widget area.
4. Insert the Sunlight API Key in the widget options
4. Enjoy!
